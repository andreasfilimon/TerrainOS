# Code of Conduct

## Our Standards

TerrainOS is a technical project. We keep it simple:

- Be direct and constructive
- Criticize code, not people
- Respect different skill levels — everyone starts somewhere
- Stay on topic

## Not Tolerated

- Harassment, personal attacks, or discrimination
- Spam or off-topic noise
- Deliberate sabotage of the project

## Reporting

Report issues to the maintainer via GitHub: [@andreasfilimon](https://github.com/andreasfilimon)

Violations may result in removal from the project.
